Đã tạo EA theo tài liệu của bạn, lot cố định 0.01, sẵn sàng test.

File EA: micro-scalper/MR_MicroScalp.mq5:1
Chiến lược: mean‑reversion micro‑scalp (fade spike)
Vào lệnh: phát hiện micro‑spike theo tick (delta ≥ IN_SPIKE_TICKS, tick‑volume > EMA*IN_VOLUME_MULT), vào ngược chiều bằng Market
Thoát lệnh: TP theo tick (mặc định 2), SL theo tick (6), giới hạn thời gian (12,000 ms), có trailing breakeven tuỳ chọn
Filter an toàn: spread tối đa + EMA spread, killzone (deny hours), ATR(M1) so với baseline, ping, dừng theo chuỗi thua, dừng theo lỗ tối đa ngày, giới hạn số lệnh đồng thời
Nhật ký: CSV mr_micro_scalper_log.csv (COMMON folder), log cả open/close, slippage, PnL, duration, retcode; in thống kê định kỳ (fill rate, avg slippage, avg PnL, max DD, consecutive losses)
Thông số chính (có thể chỉnh trong Inputs):

LOT=0.01 (cố định), TP_ticks=2, SL_ticks=6, TIME_LIMIT_MS=12000
MAX_CONCURRENT=4, SPREAD_MAX_ticks=2, SPREAD_MULT=2.0
VOLUME_MULT=2.0, SPIKE_TICKS=2
ATR: PERIOD=14, ATR_MIN_MULT=0.5, ATR_MAX_MULT=2.0
Killzone: IN_KILLZONE_DENY_HOURS (ví dụ: "0,1,2,3"), News filter (placeholder tắt mặc định)
Stop/cooldown: CONSECUTIVE_LOSS_STOP=5, COOLDOWN_MIN=30, MAX_DAILY_LOSS_USD=1000
Cài đặt/Chạy nhanh:

Mở MetaEditor → File → Open Data Folder → MQL5/Experts → copy file micro-scalper/MR_MicroScalp.mq5:1 vào đó → Compile → gắn EA lên chart XAUUSD (bất kỳ TF, EA chạy theo tick; dùng M1 cho ATR).
