# BROKER CHECKLIST (XAUUSD M1 – PTG v4)
SYMBOL: XAUUSD (Gold), DIGITS: 3 hoặc 2, POINT size = 0.01
TICK SIZE: = POINT (đa số broker: 0.01). Nếu khác, kiểm tra logs "Stops level".

[ ] Spread trung bình <= 12 pips (120 pts nếu điểm=0.01). Nếu thường >15:
    -> dùng Usecase 25 (Spread 15/12) hoặc 43 (low-vol friendly).

[ ] Stops Level <= ~200 pts (2.00$). Nếu cao hơn, vào lệnh pending có thể bị từ chối.
    -> Xem log "Invalid stops" / "Pending send failed". Nếu gặp nhiều: tăng SL_Fixed +2..4p,
       hoặc bật Market Execution (UsePendingStop=false) tạm thời.

[ ] VPS <20ms tới server broker, Mode "Every tick based on real ticks".

[ ] Kiểm tra swap/commission: Net profit trong Backtest đã +swap +commission.

[ ] Rollover / News: nếu broker giãn spread mạnh ở 00:00 hoặc NFP,
    cân nhắc bật Blackout 5’ trước / 10’ sau (Inputs hoặc Usecase tuỳ chọn).

TIP:
- Nếu log nhiều "Round number nearby": UC24 (RN 5/3) hoặc UC28 (RN OFF) hoặc UC32/44 (RN strict/asymmetric).
- Nếu log nhiều "Spread too wide": UC25 (15/12) hoặc UC34/47 (10/9 ECN).
